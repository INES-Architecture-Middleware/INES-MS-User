# INES User Microservice

This project is a user management microservice built using Node.js and Express. It provides a complete API for managing users, including functionalities for creating, logging in, logging out, retrieving, updating, and deleting user accounts. The API is designed according to the OpenAPI 3.0 specification.

## Features

- User registration
- User login and logout using JWT tokens
- Retrieve user details by ID
- Update user information
- Delete user accounts

## Technologies Used

- Node.js
- Express
- TypeScript
- JSON Web Tokens (JWT)
- bcrypt for password hashing

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm (Node package manager)

### Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd ines-ms-user
   ```

3. Install the dependencies:
   ```
   npm install
   ```

### Running the Application

1. Start the application:
   ```
   npm start
   ```

2. The server will run on `http://localhost:3000`.

### API Documentation

Refer to the OpenAPI specification for detailed information on the available endpoints and their usage.
